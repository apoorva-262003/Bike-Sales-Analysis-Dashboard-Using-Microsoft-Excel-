# Bike-Sales-Analysis-Dashboard-Using-Microsoft-Excel
Developed an interactive dashboard using Pivot Tables, Pivot Charts, Slicers, and Excel data visualization tools. Analyzed customer demographics, income levels, commute distance, education, marital status, and regional sales patterns. Identified factors influencing bike purchase decisions and customer buying behavior.
